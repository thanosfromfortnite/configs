# Configurations for NYCDOE Computers

## How to Use

Run start.bat

## Current Features

* Changing PC name and workgroup

* Change name and workgroup to nomenclature for cart laptops, smartboard laptops, teacher stations, computer lab computers

* Change date/time or force time sync

* Recreate student account

* Delete test account & user folder

* Delete account & user folder for other users

* Restart the computer

## Upcoming Features

* Windows Update options

* Postpone Windows Update indefinitely

* Run Updates
